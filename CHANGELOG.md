# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-06-30

### Changed
- Replaced Google Maps and Google Places API with free OpenStreetMap services (Nominatim and Overpass API).
- Removed all Google Maps API key requirements from the project.
- Updated UI components to handle OSM data formats.

## [1.0.0] - 2026-06-30

### Added
- Initial production-ready release of TransitGuard.
- Destination Arrival Alerts with background tracking.
- Live Location Sharing via Supabase.
- Nearby Bus Stop and Hospital discovery.
- Emergency Contact management and SOS Hub.
- Material Design 3 UI with Dark Mode support.
- Comprehensive documentation for setup, environment, and deployment.
