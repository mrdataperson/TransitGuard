# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-30

### Added
- Initial production-ready release of TransitGuard.
- Destination Arrival Alerts with background tracking.
- Live Location Sharing via Supabase.
- Nearby Bus Stop and Hospital discovery using Google Places API.
- Emergency Contact management and SOS Hub.
- Material Design 3 UI with Dark Mode support.
- Comprehensive documentation for setup, environment, and deployment.
